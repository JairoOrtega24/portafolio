Aplicación web con 3D para el museo de Guano

La aplicación web cuenta con diferentes páginas con información del museo, pero el apartado más llamativo es la integración de objetos 3D con ayuda de la librería Three.js.

Este proyecto el de elaboración propia y única y cuenta con información del museo, recorridos virtuales por las instalaciones del museo, 9 objetos del museo representados en 3D, área de contacto a través de correo electrónico y un panel de control enlazado a una base de datos con PHP desde el cual se pueden hacer todo tipo de cambios en la aplicación web sean estos grandes o pequeños.

Actualmente la aplicación web se puede observar en el dominio www.museoguano.com y los encargados del museo lo administran luego de haberse dado la entrega de mi parte.

Tienda multimedia para Android

La aplicación móvil fue desarrollada en Android Studio a través del lenguaje de programación Java, cuenta con diferentes pantallas, un menú deslizante, conexión a base de datos y la posibilidad de descargar: imágenes (Wallpapers), Videos (Fondos en movimiento) y tonos de llamada o notificación.

El diseño de la aplicación web es un desarrollo propio ya que cada uno de los puntos presentados fue elaborado únicamente para la aplicación web sean estos: logos, imágenes, fondos, etc.

Tienda e-commerce de sneakers

La tienda de sneakers se elaboró para la marca “Sneakers distributions” que vende sneaker de alta calidad, esta tienda en línea tiene diferentes funcionalidades como carrito de compras, puntuaciones por producto, comentarios por productos, pero la principal característica es que contiene un amplio catálogo de productos clasificados de forma específica con la posibilidad de comunicarse con el vendedor y poderse comprar.

La aplicación web se puede visualizar en el dominio www.sneakersdistribution.com y dicha tienda fue elaborado a través de WordPress con plugins además de edición de código.

Página web para la empresa Inlli

La página web fue diseñada con las peticiones establecidas por el dueño/creador de la marca/empresa Inlli quien necesitaba una página publicitaria de los productos ofertados.

Para llevar a cabo la creación de la página se necesitaba realizar varias imágenes publicitarias y crear videos con buena calidad que se muestran en muchos puntos.

La página web se puede encontrar en el dominio www.inlliaromasdealtura.com, fue creada con WordPress y contiene un catálogo de productos e información variada sobre el emprendimiento.

Aparte del desarrollo de software tengo conocimientos de sistemas operativos y de hardware, el mantenimiento que he realizado es principalmente sobre el software y la limpieza del hardware, por otra parte, en cuanto a la reparación he realizado cambios de memoria RAM, disco duro, actualización de BIOS y reparación de discos duros.
